# inspequte Rule Documentation

inspequte is a fast, CLI-first static analysis tool for JVM class and JAR files.
It focuses on CI/CD speed, deterministic output, and SARIF-only reporting for global
standard compatibility.

![inspequte logo](logo.png)

inspequte rules define static analysis checks for JVM class and JAR files.
Each rule describes what pattern is detected, why it matters, and how to fix it.

See the full list of rules in [Rules Index](rules/index.md).
